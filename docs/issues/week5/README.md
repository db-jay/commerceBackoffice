# Week 5 Issue Pack (Processing vs Learning)

이 디렉터리는 5주차 이슈를 `처리(구현)`와 `학습` 트랙으로 분리해 발행하기 위한 패키지다.

## 생성 이슈 구성
1. `[W5][진행관리] 5주차 실행 컨트롤타워`
2. `[W5][처리] display/event 컨텍스트 1차 구현`
3. `[W5][학습] display/event 컨텍스트와 기간 규칙 이해`

## 운영 규칙
- 처리 트랙 이슈는 코드/테스트/증빙 중심으로 완료한다.
- 학습 트랙 이슈는 체크리스트를 두지 않고, 주니어가 읽고 숙지할 핵심 내용 중심으로 작성한다.
- 이슈 생성 시 라벨을 반드시 부착한다.
  - 공통: `week:5`
  - 처리: `track:processing`
  - 학습: `track:learning`
  - 진행관리: `track:control`

## 발행 방법
```bash
bash docs/issues/week5/create_issues.sh
```
